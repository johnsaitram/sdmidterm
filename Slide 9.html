Sensor Development – GSR & Temperature System

GSR Auto-Calibration: Added automatic calibration to Galvanic Skin Response sensors to ensure consistent baseline readings regardless of environmental conditions or skin variations.

Continuous Temperature Monitoring: The system continuously reads both skin and ambient temperatures, applying the following validated research-based formula to estimate core body temperature.

coreTemp = skinTemp + α × (skinTemp – ambientTemp)
Using α = 0.3094 (Chen et al.)

void setup() {

Serial.begin(9600);

Serial.println("=== GSR Sensor Calibration Starting ===");

Serial.println("Please do NOT wear the sensor during this phase.");

long sum = 0;

// Take 100 readings to determine calibration baseline

for (int i = 0; i < 100; i++) {

sensorValue = analogRead(GSR_PIN);

sum += sensorValue;

delay(10);

}

calibrationValue = sum / 100; // average baseline

Serial.print("Calibration complete. Baseline = ");

Serial.println(calibrationValue);

Serial.println("You can now wear the sensor. Readings and graph will begin...");

delay(2000);

}
