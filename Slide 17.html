Sensor Health & Failsafe Response

Are the sensors functioning?

Failure Type What it looks like in data Example Cause
Disconnected / shorted Values stuck at 0 or 1023 (max ADC) Loose wire, broken lead
Frozen output Same reading for a long time Software lock, bus hang
Unrealistic reading Way outside human limits Noise, wrong calibration
Spiking / glitching Sudden impossible jumps Poor contact, EMI noise

What if one or more sensors fail

  • If 1 sensor fails, the others automatically share its weight.

  • If 2 fail, the remaining one still works at full weight (100%).

  • If all fail, the system enters a failsafe mode (mid-level cooling + alert).
